# Feed Calculation System - Quick Start Guide

## Getting Started in 5 Minutes

### 1. Access the Feed Calculation Features

**From Sidebar Navigation:**
- Look for "Feed Calculations" link in the sidebar menu
- Click to access the Feed Items/Calculations management page
- Or access "Feed Charts" for distribution planning

### 2. Create Your First Feed Item

**Steps:**
1. Go to Feed Calculations ‚Üí Feed Items tab
2. Fill in:
   - **Name**: e.g., "Corn"
   - **Quantity per Bag**: e.g., "50" (in kg)
   - **Price per Bag**: e.g., "100"
3. Optional fields:
   - Category: Select from dropdown (Cereals, Supplements, etc.)
   - Unit: Choose kg, lbs, g, mg, or tons
   - Supplier: Your feed supplier name
   - Shelf Life: How long the feed lasts
4. Click "Add Feed Item"

### 3. Create a Calculation

**Steps:**
1. Go to Feed Calculations ‚Üí Calculations tab
2. Fill in:
   - **Feed Item**: Select from dropdown (the item you just created)
   - **Quantity per Time**: e.g., "2.5" kg per feeding
   - **Times per Day**: e.g., "2"
   - **Number of Animals**: e.g., "50"
3. Click "Preview Calculation" to see the math:
   - Daily consumption
   - 10/20/30-day projections
   - Costs for each period
   - Bags required
4. Click "Save Calculation" to store it

### 4. Create a Feed Chart

**Steps:**
1. Go to Feed Charts page
2. Click "Create New Chart"
3. Fill in:
   - **Chart Name**: e.g., "Summer Feeding Plan"
   - **Number of Animals**: e.g., "100"
   - **Animal Type** (optional): e.g., "Dairy Cows"
4. Click "Add Feed Item" to add items to your distribution:
   - Select feed item
   - Set quantity per time (kg)
   - Set times per day
   - Repeat for each item
5. Click "Create Chart"

### 5. View & Export Results

**Viewing Calculations:**
1. Find your chart in the list
2. Click "View Calculations"
3. See cost breakdowns for:
   - Daily costs
   - 10-day costs (with bags needed)
   - 20-day costs (with bags needed)
   - 30-day costs (with bags needed)
4. See item-by-item breakdown

**Exporting:**
1. In calculations view, click:
   - "Export CSV" - for Excel/spreadsheet
   - "Export PDF" - for printing/sharing
2. File downloads automatically

## Understanding the Math

### Basic Formula
```
Daily Quantity = Quantity per Time √ó Times per Day √ó Number of Animals
Cost for 10 Days = (Daily Quantity √ó 10) √∑ Bag Size √ó Price per Bag
```

### Example Breakdown
**Feed Item:** Corn
- Size: 50kg per bag
- Price: $100 per bag

**Usage:**
- 3kg per feeding
- 2 feedings per day
- 50 animals

**Results:**
- Daily: 3 √ó 2 √ó 50 = **300 kg**
- 10-day: 300 √ó 10 = **3,000 kg**
- Bags needed: 3,000 √∑ 50 = **60 bags**
- Cost: 60 √ó $100 = **$6,000**

## Common Questions

**Q: How do I change a feed item's price?**
A: Go to Feed Items, find the item, click Edit (if available) or delete and recreate with new price.

**Q: Can I have different feeding schedules for different animals?**
A: Yes! Create separate charts for different animal types.

**Q: What if I need to plan for 45 days instead of 30?**
A: The 30-day calculation covers a month. For longer periods, multiply the 30-day cost.

**Q: Can I export multiple charts at once?**
A: Export one chart at a time. For multiple comparisons, export each and combine in Excel.

**Q: How accurate are the calculations?**
A: Calculations use exact arithmetic rounded to 2 decimal places. Bag counts use ceiling function (always rounds up to whole bags).

**Q: Can I undo a deletion?**
A: Deletions are permanent. Be careful when clicking Delete.

**Q: What's the difference between a calculation and a chart?**
A: Calculation = single item usage. Chart = combination of multiple items for a complete feeding plan.
